# Project Recipes
This is project is a part of Odin Project Foundation Curriculum and aims to improve fundamental web development skills.

# Description 
This projects showcases Burmese recipes, ingredients and ways to make a delicious ones on your own. There are three famous recipes of Burma in the website and preparations steps are listed clearly. 

# Technology
This website used HTML5 only and explored various tags of it.
